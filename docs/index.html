🔧 TOOLS

4つのMCPツール

メイン

🖼️ generate_image

テキストプロンプトから画像生成。全6モデル対応。Imagenモデルではアスペクト比指定・複数枚同時生成が可能。

編集

✏️ edit_image

既存画像 + テキスト指示で画像編集。背景変更、スタイル変換、オブジェクト追加削除等。Gemini Nativeモデルのみ。

NEW

🔀 compose_images

複数画像（2〜14枚）を合成・ブレンド・スタイル転写。顔+ヘアスタイル合成、人物+背景合成、広告素材作成等。

📋 list_models

利用可能な6モデルの一覧と特徴・価格・推奨用途を表示。モデル選択に迷った時に。

🚀 QUICK START

2ステップで始める

① Gemini API Key を取得

Google AI Studio → API Keyを作成してコピー。無料枠あり。

② MCPサーバーに接続

お使いのAIクライアントに以下を設定:

URL: https://gemini-image-mcp-two.vercel.app/api/mcp Authorization: Bearer <your-gemini-api-key>

Claude Desktop/Cursor/VS Codeの場合は README を参照
