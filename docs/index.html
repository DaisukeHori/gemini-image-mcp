🔧 TOOLS

4つのMCPツール

メイン

🖼️ generate_image

テキストプロンプトから画像生成。全6モデル対応。Imagenモデルではアスペクト比指定・複数枚同時生成が可能。

編集

✏️ edit_image

既存画像 + テキスト指示で画像編集。背景変更、スタイル変換、オブジェクト追加削除等。Gemini Nativeモデルのみ。

NEW

🔀 compose_images

複数画像（2〜14枚）を合成・ブレンド・スタイル転写。顔+ヘアスタイル合成、人物+背景合成、広告素材作成等。

📋 list_models

利用可能な6モデルの一覧と特徴・価格・推奨用途を表示。モデル選択に迷った時に。

🚀 QUICK START

始め方

① Gemini API Key を取得

Google AI Studio → API Keyを作成してコピー。

無料枠あり（約500リクエスト/日）。有料はPay-as-you-goで$0.02〜/枚。

② 下のクライアント別設定をコピーして接続

お使いのAIクライアントを選んで、設定をコピーするだけ。

🔌 INTEGRATION

クライアント別 接続設定

お使いのAIクライアントを選択して、設定をコピー。YOUR_GEMINI_API_KEY を実際のキーに置き換えてください。

🔒 SECURITY

セキュリティ

✅ HTTPS暗号化

全通信はTLSで保護。API Keyが平文で流れることはありません。

✅ ステートレス

API Keyはリクエスト処理中のみ使用。保存もログ出力もしません。

✅ ソースコード全公開

GitHub で全処理を確認できます。
